# InterviewIQ

InterviewIQ is an AI-Powered Mock Interview and Placement Preparation Platform designed to help students improve their interview skills through personalized mock interviews, resume analysis, performance tracking, and AI-generated feedback.

## Features
- AI Mock Interviews
- Resume Analysis
- Performance Dashboard
- Interview History
- Personalized Feedback
- Placement Readiness Tracking

## Project Status
Currently in Planning & UI Design Phase.

## Future Scope
- Voice-Based Interviews
- ATS Resume Scoring
- AI Career Recommendations
- Mobile Application Support
